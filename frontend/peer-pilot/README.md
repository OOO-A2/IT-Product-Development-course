# PeerPilot Frontend

Typescript + React + Vite based website.

## How to Run

Clone the repository and navigate to `peer-pilot` folder. Use one of methods below.

### Method 1: With [Node.JS](https://nodejs.org/en/download)

Run:
```sh
npm install
npm run dev
```

Open http://localhost:5173/



### Method 2: With [Docker](https://www.docker.com/)


Run:
```sh
docker compose up
```

Open http://localhost:8080/
